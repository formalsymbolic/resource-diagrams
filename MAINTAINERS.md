# Maintainers

This project is currently maintained by a small group of contributors focused on implementations (as illustrative symbolic models) of monoidal categorical methods for AI system analysis.

## Current Maintainers

- Resource Diagrams Contributors (collective)

## How to Contact Maintainers

- For security issues: See [SECURITY.md](SECURITY.md)
- For general questions and contributions: Open an issue or discussion
- For governance or scope questions: See [GOVERNANCE.md](GOVERNANCE.md)

We welcome contributions from researchers and engineers, particularly in the areas of:
- New modeling patterns that make resource and information-flow properties visible
- Improvements to diagram construction and export quality
- Strengthening structural correspondence to the source diagrammatic constructions and verification within the model

See [CONTRIBUTING.md](CONTRIBUTING.md) for details on how to get involved.
